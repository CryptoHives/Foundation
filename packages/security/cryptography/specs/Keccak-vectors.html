Keccak Test Vectors

Overview

Keccak is the original hash function family before it was standardized as SHA-3. The key difference is the padding: Keccak uses 0x01 while SHA-3 uses 0x06.

Keccak-256 is primarily used in Ethereum for:

  • Address generation
  • Transaction hashing
  • Merkle tree nodes
  • Smart contract function selectors
