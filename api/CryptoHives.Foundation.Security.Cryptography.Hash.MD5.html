Remarks

This is a fully managed implementation of MD5 based on RFC 1321. It does not rely on OS or hardware cryptographic APIs, ensuring deterministic behavior across all platforms and runtimes.

Security Warning: MD5 is cryptographically broken and should NOT be used for security purposes such as digital signatures, password hashing, or message authentication. It is provided only for legacy compatibility and non-cryptographic uses such as file checksums.
